# Red Hat System Administration II (RH134) 10.0 🚀

This repository covers advanced Linux administration tasks, focusing on scalability, security, and professional deployment techniques.

## 🎯 Advanced Goals
- [ ] **Automated Installations**: Mastering Kickstart for rapid deployments.
- [ ] **Advanced Networking**: Bonding, Teaming, and advanced Firewalling with `firewalld`.
- [ ] **Storage Mastery**: Managing Logical Volumes (LVM) and advanced file systems.
- [ ] **Security & Compliance**: Deep dive into SELinux and permissions (ACLs).
- [ ] **System Recovery**: Troubleshooting the boot process and fixing critical errors.
- [ ] **Scripting**: Automating daily tasks with Bash.

## 🛠 Advanced Command Toolkit
| Command | Description |
| :--- | :--- |
| `lsblk` | List all block devices and LVM structures |
| `getenforce` / `setenforce` | Manage SELinux modes |
| `firewall-cmd --permanent` | Persist firewall rules |
| `nmcli con mod` | Modify network connections via CLI |
